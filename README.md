# Landing demo Implantologia - Sorriso & Salute Monza

Landing page statica ottimizzata per SEO locale e campagne Google Ads su servizio di implantologia dentale, con quiz/prenotazione multistep, calendario visuale, sezione equipe e riepilogo completo per lo studio.

## Fonti contenuto

- Sito ufficiale: https://www.sorrisoesalute.it/
- Pagina implantologia: https://www.sorrisoesalute.it/attivita/implantologia/

## Deploy Vercel

Preset: Other
Build command: lasciare vuoto
Output directory: lasciare vuoto

Il progetto e composto da file statici e non richiede installazione di dipendenze. Il quiz genera una mail precompilata a `info@sorrisoesalute.it` tramite `mailto:`, includendo problema, obiettivo, dubbi, tempistica, preferenza appuntamento e contatti.
